# 📊 Data Analytics & Machine Learning Projects

This repository contains a collection of Data Analytics and Machine Learning projects completed as part of my internship. These projects focus on data preprocessing, analysis, visualization, and predictive modeling.

---

## 🚀 Projects Included

### 🔹 1. Customer Churn Prediction
- 📌 Objective: Predict whether a customer will churn or not
- 🛠️ Tools: Python, Pandas, NumPy, Scikit-learn
- 📊 Techniques:
  - Data Cleaning
  - Encoding & Feature Engineering
  - Logistic Regression, Decision Tree
- 🎯 Outcome:
  - Identified key factors affecting customer churn
  - Built classification models for prediction

---

### 🔹 2. Titanic Survival Prediction
- 📌 Objective: Predict survival of passengers
- 🛠️ Tools: Python, Pandas, Seaborn, Scikit-learn
- 📊 Techniques:
  - Missing value handling
  - Feature selection
  - Classification models
- 🎯 Outcome:
  - Built predictive model for survival
  - Performed EDA to understand patterns

---

### 🔹 3. Sales & Profit Analysis Dashboard (Power BI)
- 📌 Objective: Analyze sales performance and profitability
- 🛠️ Tools: Power BI
- 📊 Features:
  - Total Sales & Profit KPI cards
  - Sales by Country & Product
  - Monthly Trend Analysis
  - Profit vs Loss visualization
- 🎯 Outcome:
  - Created interactive dashboard
  - Derived business insights for decision-making
